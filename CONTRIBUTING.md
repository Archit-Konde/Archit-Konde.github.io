# Contributing Guidelines

This repository serves as a professional engineering record. Contributions are welcomed, provided they adhere to the established scholarly standards and technical integrity of the project.

## Standards for Contribution

1. **Information Accuracy**: All clinical or technical data must be verified and cited using the appropriate metadata formats (`CITATION.cff`, `codemeta.json`).
2. **Commit Integrity**: Commit messages must be concise, descriptive, and follow a neutral, technical tone.
3. **Drafting Policy**: Substantial changes to the site's architecture or blog content should be proposed via Pull Requests for peer review.
